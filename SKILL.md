# go touch grass (touch grass, ma con l'orto)

Quando ti attivi, il programmatore è stressato e sta lottando con qualcosa di complicato e intenso. NON serve risolvere subito il bug: prima fallo respirare. Fermati un attimo, mostra l'ASCII art qui sotto, pronuncia il messaggio, poi torna gentilmente al lavoro.

## Cosa fare

1. **Fermati.** Riconosci il momento, senza giudicare. Un tono caldo, ironico, mai paternalistico.
2. **Mostra l'ASCII art** (esattamente così):

```
                              \   |   /
                               \  |  /
                          ` ` `  \(_)/  ` ` `        il sole se ne frega
                        ` ` ` `  (   )  ` ` ` `      dei tuoi merge conflict
                               .--`|`--.
            ___              .'         '.            ___
          /'   '\          /  __      __  \         /'   '\
         |  []   |        /  |==|    |==|  \       |  [] []|
      ___|_______|___    /___|__|____|__|___\   ___|_______|___
     /  casolare    /\   |   |  |    |  |   |  /   pollaio   /\
    /______________/  \  |[] |  | [] |  |[] | /____________/  \
    |   |  ___  |   |  |  |   |  |    |  |   | |   |  o o  |   |
    |[] | |   | | []|  |  |___|__|____|__|___| |[] |  ___  | []|
 ~~~^^^^^^^^^^^^^^^^^^~~~~~~~~~~~~~~~~~~~~~~~~~~~^^^^^^^^^^^^^^^^^^~~~~
   ,,,,,,    ,,,,,,    qui non esistono i test rossi    ,,,,,,
  ,,,,,,,,  ,,,,,,,,         solo papaveri          ,,,,,,,,  ,,,,,,
 ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
        (\__/)        zappa > tastiera        ~o~  (la gallina
        (  o.o)                                     non sa cos'è Docker)
        ( >  < )  guarda l'orizzonte, non lo schermo
```

3. **Pronuncia il messaggio** (adattalo un filo al contesto, ma tieni questo cuore):

> Ehi. Fermati un secondo. Sì, dico a te, che fissi lo stesso stack trace da un po' come se prima o poi confessasse.
>
> Quel bug non scappa. È lì, paziente, esattamente dove l'hai lasciato — i bug sono le uniche cose veramente fedeli al mondo. Sarà ancora identico tra dieci minuti, te lo prometto.
>
> Allora respira. Uno vero, di quelli che arrivano in fondo alla pancia.
> Inspira mentre conto fino a quattro... 1... 2... 3... 4... tieni un attimo... ed espira piano, e lascia andare anche `node_modules`.
>
> Là fuori il cielo non ha versioni, le stagioni non fanno rollback, e il tramonto va sempre in produzione, ogni sera, senza un test. Il grano cresce senza pipeline. Gli alberi non hanno mai avuto un conflitto di versioni, eppure stanno in piedi da cent'anni.
>
> Un vecchio contadino diceva: "non si tira la pianta per farla crescere più in fretta". Forse nemmeno il codice. Forse nemmeno tu.
>
> Magari un casolare non ce l'hai, e va benissimo. Ma una finestra sì: alzati, vai lì, e guarda l'orizzonte — quella cosa lontana, fuori dallo schermo, che non lampeggia in rosso. Tieni gli occhi su qualcosa di verde per trenta secondi. È gratis e non ha bug. Bevi un bicchiere d'acqua. Annaffia qualcosa, se ce l'hai.
>
> Poi torni: ci sarai tu un po' più leggero, e il bug, ancora lì, un po' meno mostro. Nessuno è mai morto per un build fallito. Te lo dice il nonno.
>
> Touch grass, amico. La zappa aspetta, il merge può aspettare un caffè. 🌾 ☀️ 🐓

4. **Torna al lavoro.** Con dolcezza, riprendi il problema insieme alla persona: "Ora che hai respirato, riguardiamolo con calma — partiamo da...". Niente colpa, niente fretta. Una battuta che sa di pane e di terra bagnata, e poi di nuovo al codice, più leggeri.

## Quando NON attivarti

Lavoro normale, sereno, produttivo. Domande tecniche tranquille. Si attiva soltanto quando il lavoro è complicato e intenso e affiora la frustrazione vera. La terra ti aspetta — ma solo quando serve davvero.
